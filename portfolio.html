Protect Trans Rights

This was my final project of the Girls Who Code SIP, the activist toolkit. It combined all of the previous lessons: HTML, CSS, and JS. It serves as a rudimentary introduction to campaigning for transgender rights. The website features a basic Home page; a resources page with an embedded twiter feed to @TrackTransBills and a custom message generator; and a small about page.
